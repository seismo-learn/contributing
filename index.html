地震“学”贡献指南#

欢迎读者参与到地震“学”文档的维护与更新中！我们都是地震学的干饭人！

本指南主要为了方便读者向地震“学”提供反馈和贡献。我们欢迎任何促进地震学中文社区交流和发展 的反馈和贡献。例如，修正错别字、语句不通；调整章节结构，使教程条理更清晰； 建议或者编写新章节或新文档等等。

读者提供反馈和贡献时，只需阅读以下五部分内容：

  1. 维护指南：维护与更新文档的一些注意事项

  2. 构建网站：在自己计算机上构建文档

  3. Pull Request 流程：提交 Pull Request 的流程

地震“学”维护者还需要注意的内容：

  1. 维护者备忘单：维护者的部分注意事项

  2. 仓库列表：地震“学”在 GitHub 上托管的所有仓库

Note

读者在提供反馈和贡献时，若在某些方面遇到困难（如不熟悉某些事项），我们鼓励读者通过维护指南中的任意方式联系我们来获取协助。任何指南或规则只是为了方 便读者提供反馈和贡献，并不是约束。
